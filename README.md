# Veris

Veris é um app Flutter para registrar e acompanhar a rotina de cuidados do bebê de forma simples, rápida e organizada.

O objetivo do app é ajudar pais e cuidadores a manterem um histórico claro das principais atividades do dia, como mamadas, trocas de fralda, sono e outros registros importantes. A tela inicial reúne ações rápidas para novos registros e uma linha do tempo com as atividades recentes.

## Funcionalidades

- Registro rápido de atividades do bebê.
- Visualização da última atividade registrada.
- Histórico em linha do tempo.
- Interface simples, limpa e focada no uso diário.
- Persistência local dos registros no dispositivo.

## Tecnologias

- Flutter
- Dart
- Shared Preferences para armazenamento local

## Como executar

```bash
flutter pub get
flutter run
```
